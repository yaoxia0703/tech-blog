# プロジェクト

## manpower-blog

**Spring Boot 3** と **Vue 3** を用いて開発している、企業向け構成を意識したブログシステムです。  
本プロジェクトでは、モダンなバックエンドアーキテクチャ、セキュリティ設計、および拡張性の高いシステム開発の実践を目的としています。

---

### 技術スタック

**バックエンド**

- Java 21
- Spring Boot 3
- Spring Security
- MyBatis Plus
- Redis
- JWT 認証

**フロントエンド**

- Vue 3
- TypeScript
- Element Plus

**データベース**

- MySQL

---

### アーキテクチャ

バックエンドは、保守性と拡張性を高めるために、ドメイン駆動設計（DDD）の考え方を取り入れた**モジュラーアーキテクチャ**で設計しています。

主要モジュールは以下の通りです。

- **blog-common**  
  共通ユーティリティ、定数、共有コンポーネントを配置するモジュール。

- **blog-framework**  
  Security 設定、JWT 認証、グローバル例外処理、MyBatis 設定などのインフラ・共通基盤を担うモジュール。

- **blog-module-system**  
  ユーザー管理、ロール管理、権限制御、RBAC 認可など、システム領域を担当するモジュール。

- **blog-module-content**  
  記事、カテゴリ、タグ、コメント、ユーザー操作など、コンテンツ領域を担当するモジュール。

- **blog-admin-api**  
  管理画面向けのバックエンド API を提供するモジュール。

- **blog-portal-api**  
  ブログのフロント画面向けの公開 API を提供するモジュール。

---

### セキュリティ設計

- Spring Security ベースの認証
- JWT トークンによるステートレス認可
- RBAC（ロールベースアクセス制御）
- API 単位の権限検証

---

### API 設計

本システムは **RESTful API 設計原則** に従って構築しています。

主な設計方針は以下の通りです。

- 統一レスポンス構造（`Result<T>`）
- 一貫した例外・エラーハンドリング
- 管理 API と公開 API の明確な分離
- 標準化されたリクエスト / レスポンス形式

API の詳細設計については **api-design.md** に整理しています。
